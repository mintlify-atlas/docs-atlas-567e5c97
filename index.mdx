## What is FrankerFaceZ?

FrankerFaceZ (FFZ) is a browser extension and userscript that enhances the Twitch experience. It provides:

- **Custom emotes** — thousands of emotes from channel and global emote sets, with animation and modifier support
- **Custom badges** — per-channel and global badge support with flexible rendering
- **Chat improvements** — rich content rendering, link cards, viewer cards, and advanced tokenization
- **Player enhancements** — theater mode improvements, player controls, and customization
- **Theme system** — full CSS customization with live preview
- **Add-on framework** — a robust module and settings API for third-party extensions

<Info>
FrankerFaceZ is actively developed and open source under the Apache 2.0 license. Contributions are welcome!
</Info>

## For Developers

FrankerFaceZ exposes a full JavaScript API for building add-ons. Every feature in FFZ itself is built on the same module system available to add-on developers.

```typescript
import { Addon } from 'utilities/addon';

export default class MyAddon extends Addon {
  static info = {
    id: 'my-addon',
    name: 'My Addon',
    version: '1.0.0',
    description: 'A custom FrankerFaceZ add-on',
    author: 'Your Name'
  };

  async onEnable() {
    // Register a setting
    this.settings.add('my-addon.show-widget', {
      default: true,
      ui: {
        path: 'Add-Ons > My Addon >> General',
        title: 'Show Widget',
        component: 'setting-check-box'
      }
    });

    this.log.info('My addon is enabled!');
  }
}

Addon.register(MyAddon);
```
